# range
Javascript version of Python range

### Install
```
npm install rangejs
```

##### Add to your project
```
const range = require('rangejs');
```

```
import range from 'rangejs';
```
### Usage
```
range(stop)
range(start, stop, [step])
```

### Examples
```
range(5)      // [0,1,2,3,4]
```

```
range(1,5)    // [1,2,3,4]
```

```
range(0)      // []
```

```
range(-5,0)   // [-5, -4, -3, -2, -1]
```

```
range(1,5,2)  // [1,3]
```
